## Documentation

For the full documentation, go to [saplingjs.com/docs](https://saplingjs.com/docs/).

## Quick installation

1. Make sure you have Node.js 10 or later.
2. Run `npm i -g @sapling/cli && sapling create`

## Ecosystem

Project                                                                                   | Description
